collagen
========

Collagen uses image segmentation algorithms to make collages

**Examples** 
http://notidaho.github.io/2013/03/30/img_cuts/

**Requirements**
--------
 - python 3  
 - pyyaml  
 - scipy   
 - numpy  
 - scikit-image  
 - Pillow  

**Usage**  
--------
python3 collagen.py [source images folder] [output folder]
